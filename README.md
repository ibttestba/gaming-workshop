# Gaming Workshop Management System

A complete centralized system for managing gaming workshops with:

- Public registration website
- Desktop session manager for gaming PCs  
- Real-time admin dashboard
- Centralized database with Supabase

## URLs

- Registration: https://your-site.netlify.app/registration/
- Admin Dashboard: https://your-site.netlify.app/admin/
- Session Manager: Download session-manager.html to gaming PCs

## Features

- Real-time session tracking
- Crash and issue reporting
- Slot management (max 4 per slot)
- One game per session enforcement
- Live admin monitoring
- Data export capabilities

## Setup

1. Configure Supabase credentials in all HTML files
2. Deploy to Netlify
3. Copy session-manager.html to gaming PCs
